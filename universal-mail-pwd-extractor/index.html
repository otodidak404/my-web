Pengekstrak Email dan Password Universal

Mode Sederhana

Mendeteksi email dan password secara otomatis dalam berbagai format, berusaha mengekstrak kombinasi email dan password dari teks.

Mode Lanjutan

Konfigurasikan sendiri pola pencarian dan format output sesuai kebutuhan Anda.

Pemisah Input:
Pemisah antar kolom dalam teks input
Posisi Email:
Posisi kolom email setelah pemisahan (1, 2, 3, dst)
Posisi Password:
Posisi kolom password setelah pemisahan (1, 2, 3, dst)
Format Output:
Gunakan {email} dan {password} sebagai placeholder

Tentang Alat Ini

Alat pengekstrak email dan password universal ini dirancang untuk membantu Anda mengekstrak kombinasi email dan password dari berbagai format teks.

Mode Sederhana

Mode ini mencoba mendeteksi email dan password secara otomatis menggunakan berbagai pola umum yang biasa digunakan:

  • Format dengan pemisah "|": ... | email@domain.com | password | ...
  • Format dengan titik dua: email@domain.com:password
  • Format dengan spasi: email@domain.com password
  • Format dengan label: email=user@example.com password=myP@ss123

Mode Lanjutan

Mode ini memungkinkan Anda mengonfigurasi pengekstrak sesuai format teks Anda:

  • Tentukan pemisah yang digunakan dalam teks ("|", ":", dll.)
  • Tentukan posisi kolom email dan password
  • Kustomisasi format output sesuai kebutuhan

Tips Penggunaan

  • Jika mode sederhana tidak berhasil mengekstrak data dengan benar, coba gunakan mode lanjutan.
  • Pastikan teks input Anda konsisten dalam penggunaan pemisah.
  • Untuk format yang sangat khusus, gunakan mode lanjutan dan sesuaikan pengaturannya.
